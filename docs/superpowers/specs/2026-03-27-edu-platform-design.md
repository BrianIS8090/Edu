# EduPlatform — Спецификация дизайна

## Обзор

Обучающая платформа для инженеров компании Group 7Lamp. Единая точка входа для всех обучающих материалов: Markdown-уроки, интерактивные HTML-модули, видео, PDF, презентации.

## Ключевые принципы

- Контент добавляется через git push — платформа подхватывает автоматически
- Без фреймворков, без бэкенда, без авторизации
- Markdown-файлы с frontmatter как основной формат контента
- Клиентский рендеринг (marked.js)
- Единый каталог — уроки и интерактивные модули в одном списке

## Структура файлов

```
Edu/
├── index.html                  # Единая точка входа (SPA)
├── assets/
│   └── <название-урока>/       # Папка с материалами урока
│       ├── video.mp4
│       ├── schema.pdf
│       └── photo.png
├── lessons/
│   └── <название-урока>.md     # Markdown с frontmatter
├── modules/
│   └── <название-модуля>/
│       ├── index.html           # Интерактивный HTML-модуль
│       └── meta.md              # Frontmatter + описание
├── build/
│   └── catalog.json             # Автогенерируемый индекс
├── scripts/
│   └── build-catalog.sh         # Скрипт сборки индекса
└── .github/
    └── workflows/
        └── deploy-pi.yml        # CI/CD (+ шаг сборки catalog.json)
```

## Формат Markdown-урока

```markdown
---
title: Основы ШИМ-диммирования
category: Светотехника
tags: [диммирование, ШИМ, LED]
---

# Содержимое урока

Стандартный GFM Markdown. Рендерится 1:1 как в редакторе.
Ссылки на материалы: [схема](../assets/pwm-basics/schema.pdf)
```

## Формат meta.md для HTML-модулей

```markdown
---
title: Калькулятор радиатора
category: Светотехника
tags: [тепло, радиатор, LED]
---

Интерактивный калькулятор для расчёта площади радиатора LED.
```

## Формат catalog.json

```json
{
  "lessons": [
    {
      "id": "pwm-basics",
      "title": "Основы ШИМ-диммирования",
      "category": "Светотехника",
      "tags": ["диммирование", "ШИМ"],
      "type": "lesson",
      "file": "lessons/pwm-basics.md",
      "assets": "assets/pwm-basics/"
    }
  ],
  "modules": [
    {
      "id": "heatsink-calc",
      "title": "Калькулятор радиатора",
      "category": "Светотехника",
      "tags": ["тепло", "LED"],
      "type": "module",
      "path": "modules/heatsink-calc/"
    }
  ]
}
```

## Механика работы

### При git push (CI/CD)

1. Скрипт `build-catalog.sh` сканирует `lessons/*.md` и `modules/*/meta.md`
2. Парсит frontmatter из каждого файла (title, category, tags)
3. Генерирует `build/catalog.json`
4. Деплоит всё на Pi5 через rsync

### При открытии сайта (клиент)

1. `index.html` загружается
2. JS запрашивает `build/catalog.json`
3. Строит sidebar с категориями и список материалов
4. При клике на урок (type: lesson) — загружает Markdown-файл через fetch, рендерит через marked.js
5. При клике на модуль (type: module) — загружает HTML-модуль в iframe

## UI и навигация

### Раскладка

- **Хедер** (52px) — логотип EduPlatform (иконка Lucide `book-open`), фиксирован сверху
- **Sidebar** (260px) — фиксированный слева, содержит категории и список материалов. Остаётся видимым всегда, включая страницу урока
- **Контент** — центрирован в оставшемся пространстве, max-width 760px

### Sidebar

- Секция "Все материалы" — показывает общее количество
- Категории группируют уроки и модули вместе
- Активный элемент подсвечивается (фон #f0f0ff, цвет #4f46e5, border-left)
- Иконка `file-text` для Markdown-уроков, `app-window` для HTML-модулей

### Список материалов (главная)

- Карточки в вертикальном списке
- Каждая карточка: иконка (цветной фон), заголовок, описание, теги
- Тег типа: "урок" (синий #e0e7ff/#4f46e5) или "интерактив" (зелёный #ecfdf5/#059669)

### Страница урока (Markdown)

- Хлебная крошка "Все материалы" для возврата
- Заголовок, теги
- Отрендеренный Markdown (заголовки, списки, ссылки, картинки, блоки кода)
- Sidebar остаётся видимым

### Страница модуля (HTML)

- Хлебная крошка "Все материалы" для возврата
- Заголовок, теги
- iframe с HTML-модулем (width: 100%, height: 80vh)
- Sidebar остаётся видимым

## Визуальный стиль

- **Тема:** светлая
- **Фон страницы:** #f5f5f5
- **Фон карточек/sidebar:** #fff
- **Границы:** #e5e7eb
- **Акцентный цвет:** #4f46e5 (индиго)
- **Шрифт:** системный (-apple-system, Segoe UI, sans-serif)
- **Шрифт кода:** JetBrains Mono
- **Иконки:** Lucide (CDN)
- **Скругления:** 8px (карточки), 12px (теги), 6px (sidebar)

## Обработка ошибок

- **Пустая категория** — не отображается в sidebar
- **Битый/отсутствующий frontmatter** — файл попадает в категорию "Без категории"
- **Битые ссылки на assets** — placeholder "Файл не найден"
- **HTML-модуль без meta.md** — отображается с именем файла как заголовком, без тегов

## Технологический стек

- **HTML/CSS/JS** — без фреймворков
- **marked.js** — рендеринг Markdown (CDN)
- **Lucide** — иконки (CDN)
- **Nginx** — веб-сервер на Pi5
- **GitHub Actions** — CI/CD с шагом сборки catalog.json
- **bash-скрипт** — парсинг frontmatter и генерация catalog.json

## Что НЕ входит в скоуп

- Авторизация и учётные записи
- Поиск
- Прогресс прохождения
- Тёмная тема
- Мобильная адаптация (может быть добавлена позже)
