# Сертификация и испытания светильников

Без сертификации или декларирования светильник нельзя продавать на территории ЕАЭС. Этот урок — практическое руководство по процедуре, документам и требованиям для вывода светильника на рынок.

---

## 1. Нормативная база

### Основные технические регламенты

| Регламент | Полное название | Что регулирует |
|---|---|---|
| ТР ТС 004/2011 | О безопасности низковольтного оборудования | Электробезопасность (основной для светильников) |
| ТР ТС 020/2011 | Электромагнитная совместимость | ЭМС (помехи и устойчивость) |
| ТР ЕАЭС 037/2016 | Об ограничении применения опасных веществ | RoHS (свинец, ртуть, кадмий и др.) |
| ТР ЕАЭС 043/2017 | О требованиях к средствам обеспечения пожарной безопасности | Для аварийных/пожарных светильников |

### Форма подтверждения соответствия

| Тип продукции | Форма подтверждения |
|---|---|
| Светильники бытовые (до 1000 В) | Декларация о соответствии |
| Светильники для взрывоопасных зон | Сертификат соответствия |
| Аварийные светильники (пожарные) | Сертификат по ТР ЕАЭС 043 |
| Драйверы / блоки питания | Декларация о соответствии |

**Важно:** Для большинства обычных светильников достаточно **декларации** (не сертификата). Сертификат обязателен только для взрывозащищённых и пожарных светильников.

---

## 2. Процедура декларирования

### Пошаговый план

```
1. Подготовка → 2. Испытания → 3. Оформление → 4. Регистрация → 5. Маркировка
```

### Шаг 1: Подготовка документации

**Что нужно подготовить:**

| Документ | Описание |
|---|---|
| Техническое описание | Характеристики, конструкция, материалы |
| Руководство по эксплуатации | Установка, подключение, обслуживание |
| Чертежи / 3D-модели | Конструкция светильника |
| Спецификация комплектующих | LED-модули, драйверы, корпус |
| Технические условия (ТУ) или ГОСТ | Документ, по которому производится изделие |
| Контракт + инвойс (для импорта) | Для импортной продукции |

### Шаг 2: Испытания в аккредитованной лаборатории

Лаборатория должна быть аккредитована в национальной системе (ФСА — Росаккредитация).

**Основные испытания:**

| Испытание | ГОСТ | Что проверяют |
|---|---|---|
| Электробезопасность | ГОСТ IEC 60598-1 | Изоляция, заземление, защита от поражения |
| Тепловые испытания | ГОСТ IEC 60598-1 | Нагрев при нормальной и аномальной работе |
| Защита IP | ГОСТ 14254 (IEC 60529) | Пыле- и влагозащита |
| Защита IK | ГОСТ IEC 62262 | Ударопрочность |
| ЭМС (излучение) | ГОСТ CISPR 15 | Уровень электромагнитных помех |
| ЭМС (устойчивость) | ГОСТ IEC 61547 | Устойчивость к внешним помехам |
| Фотометрия | ГОСТ Р 54350 | Световой поток, КСС, эффективность |
| Пульсация | ГОСТ Р 54945 | Коэффициент пульсации (≤10%, для офисов ≤5%) |
| RoHS | ГОСТ IEC 63000 | Содержание опасных веществ |

### Шаг 3: Оформление декларации

**Варианты схем декларирования:**

| Схема | Описание | Для кого |
|---|---|---|
| 1д | Испытания образцов + производственный контроль | Серийное производство |
| 2д | Испытания образцов | Партия |
| 3д | Испытания единичного изделия | Единичное изделие |
| 4д | Испытания + сертификация СМК (ISO 9001) | Серийное, повышенная надёжность |
| 6д | На основе сертификата СМК | Крупные производители |

**Для большинства производителей:** схема **1д** — испытания образцов + собственный контроль.

### Шаг 4: Регистрация в реестре

Декларация регистрируется в Едином реестре ЕАЭС через сервис ФСА (fsa.gov.ru). Срок действия — до 5 лет.

### Шаг 5: Маркировка

После получения декларации на изделие и упаковку наносится знак обращения:

```
EAC
```

(Евразийское соответствие — единый знак для ЕАЭС)

## 3. Требования к испытаниям — подробно

### Электробезопасность (ГОСТ IEC 60598-1)

| Проверка | Требование |
|---|---|
| Электрическая прочность изоляции | 1500 В AC (класс I), 3750 В AC (класс II) |
| Сопротивление изоляции | ≥ 2 МОм (при 500 В DC) |
| Ток утечки | ≤ 1 мА (класс I), ≤ 0,5 мА (класс II) |
| Заземление | Сопротивление ≤ 0,5 Ом (класс I) |
| Нагрев | Температура обмоток, изоляции, пластика — не выше допустимых |

### Тепловые испытания

Светильник работает в нормальном режиме до установления теплового равновесия (обычно 4–8 часов). Измеряются:

- Температура корпуса в самых горячих точках
- Температура LED-модуля (должна быть ≤ Tj max по даташиту)
- Температура драйвера в точке Tc
- Температура проводов и клемм

### Испытания IP (ГОСТ 14254)

| Степень | Испытание |
|---|---|
| IP20 | Защита от пальцев (∅12 мм) |
| IP44 | Брызги воды со всех направлений |
| IP54 | Пыль (ограниченное проникновение) + брызги |
| IP65 | Пылезащищённый + струи воды (сопло 6,3 мм) |
| IP66 | Пылезащищённый + сильные струи (сопло 12,5 мм) |
| IP67 | Пылезащищённый + кратковременное погружение (1 м, 30 мин) |
| IP68 | Пылезащищённый + длительное погружение (глубина по ТУ) |

### Фотометрические испытания

Проводятся на гониофотометре:

| Параметр | Что измеряется |
|---|---|
| Световой поток (лм) | Общее количество света |
| Световая отдача (лм/Вт) | Эффективность |
| КСС (кривая силы света) | Распределение света в пространстве |
| Цветовая температура (К) | Оттенок белого |
| Индекс CRI (Ra) | Качество цветопередачи |
| Коэффициент пульсации (%) | Мерцание |
| Результат | Файлы .ies и .ldt |

## 5. Маркировка светильника

### Обязательная информация на светильнике (ГОСТ IEC 60598-1)

| Параметр | Пример |
|---|---|
| Наименование / модель | LINE-1200-40-4000K |
| Торговая марка | 7Lamp |
| Напряжение питания | 220–240 В AC |
| Мощность | 40 Вт |
| Класс защиты | I или II (◻ в ◻) |
| Степень IP | IP40 |
| Знак EAC | EAC |
| Страна-изготовитель | Россия |

### Маркировка на упаковке

Дополнительно к маркировке на изделии:

- Штрих-код / артикул
- Цветовая температура (3000К / 4000К / 5000К)
- Световой поток (лм)
- Индекс CRI (Ra)
- Габаритные размеры
- Масса
- Комплектность

## 7. Сроки и стоимость

### Ориентировочные сроки

| Этап | Срок |
|---|---|
| Подготовка образцов и документации | 1–2 недели |
| Испытания в лаборатории | 2–4 недели |
| Оформление декларации | 1–3 дня |
| Регистрация в реестре | 1–3 дня |
| **Итого** | **4–8 недель** |

### Что влияет на стоимость

- Количество моделей (каждая — отдельные испытания, или серия по ТУ)
- Объём испытаний (только электробезопасность или полный комплекс)
- Срочность
- Наличие готовой документации (ТУ, паспорт, РЭ)

## 8. Типичные ошибки

| Ошибка | Последствия | Решение |
|---|---|---|
| Продажа без декларации | Штраф, конфискация, запрет продаж | Пройти декларирование до начала продаж |
| Нет маркировки EAC | Штраф при проверке | Нанести на изделие и упаковку |
| Испытания в неаккредитованной лаборатории | Декларация недействительна | Проверить аккредитацию на fsa.gov.ru |
| Изменение конструкции без переиспытаний | Несоответствие протоколу | При существенных изменениях — повторные испытания |
| Нет руководства по эксплуатации | Замечание при испытаниях | Подготовить РЭ до испытаний |
| Просроченная декларация | Незаконная продажа | Продлить/обновить до истечения срока |
| Класс II без двойной изоляции | Провал испытания на электрическую прочность | Обеспечить реальную двойную изоляцию |

## 9. Чек-лист перед подачей на испытания

- [ ] Техническое описание изделия (модель, характеристики, конструкция)
- [ ] Руководство по эксплуатации (на русском языке)
- [ ] Технические условия (ТУ) или ссылка на ГОСТ
- [ ] Чертежи и схема подключения
- [ ] Образцы (обычно 2–3 шт.)
- [ ] Даташиты на комплектующие (LED, драйвер)
- [ ] Сертификаты на комплектующие (драйвер, LED-модуль) — если есть
- [ ] Фотографии изделия (общий вид, маркировка, клеммы)
- [ ] Информация о производителе (реквизиты, ИНН, ОГРН)
- [ ] Контракт + инвойс (для импортной продукции)

## Итоги

- **Декларация** — основная форма для обычных светильников; **сертификат** — для взрывозащищённых и пожарных
- Регламенты: **ТР ТС 004** (безопасность) + **ТР ТС 020** (ЭМС) + **ТР ЕАЭС 037** (RoHS)
- Испытания — только в **аккредитованных лабораториях** (проверить на fsa.gov.ru)
- Маркировка **EAC** — обязательна на изделии и упаковке
- Срок: **4–8 недель** от подготовки до получения декларации
- При изменении конструкции — проверить, нужны ли повторные испытания
