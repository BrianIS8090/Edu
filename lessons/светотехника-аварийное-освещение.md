# Аварийное и эвакуационное освещение

Аварийное освещение — обязательный элемент безопасности любого общественного и производственного здания. Ошибки в проектировании аварийного освещения — прямой риск для жизни людей и причина отказа в экспертизе. Этот урок — полный обзор норм, типов систем и практики проектирования.

---

## 1. Классификация аварийного освещения

### Общая структура (по ГОСТ Р 55842)

```
Аварийное освещение
├── Эвакуационное освещение
│   ├── Освещение путей эвакуации
│   ├── Освещение зон повышенной опасности (антипаниковое)
│   └── Освещение мест размещения средств пожаротушения и безопасности
└── Резервное освещение
    └── Для продолжения работы при отключении рабочего освещения
```

### Эвакуационное vs Резервное

| Параметр | Эвакуационное | Резервное |
|---|---|---|
| Цель | Безопасная эвакуация | Продолжение работы |
| Время работы | ≥ 1 ч (типично 3 ч) | Зависит от задачи |
| Уровень освещённости | 1–15 лк (по нормам) | 5–100% от рабочего |
| Обязательность | Да (для всех общественных зданий) | По проекту |
| Включение | Автоматическое при отключении сети | Автоматическое |

## 2. Нормативные требования

### Основные документы

| Документ | Содержание |

| СП 52.13330.2016 | Естественное и искусственное освещение (актуальная редакция) |
| ГОСТ Р 55842-2013 | Аварийное освещение (гармонизирован с EN 1838) |
| ГОСТ Р 56852-2016 | Светильники для аварийного освещения |
| Федеральный закон №123-ФЗ | Технический регламент о пожарной безопасности |
| СП 1.13130 | Эвакуационные пути и выходы |
| СП 3.13130 | Система оповещения и управления эвакуацией (СОУЭ) |
| СП 6.13130 | Электрооборудование (требования к кабелям) |

### Нормы освещённости

| Зона | Освещённость | Равномерность | Время работы |
|---|---|---|---|
| Пути эвакуации (центральная полоса) | ≥ 1 лк (на полу) | U₀ ≥ 0,4 | ≥ 1 ч |
| Зоны повышенной опасности | ≥ 15 лк (или 10% от рабочего) | U₀ ≥ 0,1 | Время действия фактора опасности |
| Антипаниковое (большие площади) | ≥ 0,5 лк | U₀ ≥ 0,4 | ≥ 1 ч |
| Места средств пожаротушения | ≥ 5 лк | — | ≥ 1 ч |

### Время включения

| Тип | Время включения |
|---|---|
| Пути эвакуации | ≤ 5 сек (50% яркости), ≤ 60 сек (100%) |
| Зоны повышенной опасности | ≤ 0,5 сек |
| Резервное освещение | ≤ 15 сек (или ≤ 0,5 сек при опасности) |

## 3. Типы систем аварийного освещения

### Автономные светильники (с встроенной батареей)

Каждый светильник содержит собственный аккумулятор.

**Плюсы:**
- Простота монтажа (нет централизованной системы)
- Работает при повреждении любой линии
- Не требует отдельного щита

**Минусы:**
- Необходимость обслуживания каждого светильника (замена батарей)
- Батарея увеличивает размер и вес светильника
- Сложнее централизованный мониторинг

**Типы автономных светильников:**

| Режим | Описание |
|---|---|
| Непостоянного действия (Non-Maintained) | В нормальном режиме — выключен. Включается только при пропадании сети. |
| Постоянного действия (Maintained) | Работает постоянно (от сети и от батареи). При отключении сети — переход на батарею. |
| Комбинированный | Содержит лампу рабочего и лампу аварийного освещения в одном корпусе. |

### Централизованная система (ЦБП / Central Battery)

Единый блок аккумуляторов питает все аварийные светильники.

**Плюсы:**
- Централизованный мониторинг и обслуживание
- Светильники компактнее (без батарей)
- Легче контролировать состояние системы
- Длительное время автономии (легко наращивается)

**Минусы:**
- Требуется отдельное помещение для ЦБП
- Огнестойкая кабельная линия от ЦБП до светильников
- Высокая стоимость начальной установки

### Сравнение систем

| Критерий | Автономные | ЦБП |
|---|---|---|
| Объект до 50 светильников | Оптимально | Избыточно |
| Объект 50–200 светильников | Допустимо | Рекомендуемо |
| Объект >200 светильников | Сложно обслуживать | Оптимально |
| Стоимость (малый объект) | Ниже | Выше |
| Стоимость (большой объект) | Выше (обслуживание) | Ниже |
| Надёжность | Независимость точек | Единая точка отказа |

## 4. Указатели и знаки безопасности

### Типы указателей

| Тип | Описание | Размер | Пример |
|---|---|---|---|
| Указатель «Выход» | Над каждым эвакуационным выходом | По расстоянию обзора | EXIT / ВЫХОД |
| Направляющий знак | Указывает направление движения | По расстоянию обзора | Стрелка + бегущий человек |
| Знак «Огнетушитель» | У средств пожаротушения | 150×150 мм и более | F04 по ГОСТ |
| Знак «Пожарный кран» | У пожарных кранов | 150×150 мм и более | F02 по ГОСТ |

### Расстояние распознавания

Размер знака зависит от дистанции, с которой он должен быть виден:

```
d = s × p

d — расстояние распознавания, м
s — высота пиктограммы, м
p — коэффициент (200 для внешне освещённых, 100 для внутренне освещённых)
```

**Пример:** Знак «Выход» с высотой пиктограммы 15 см, с внутренней подсветкой:
```
d = 0,15 × 100 = 15 м
```

### Размещение указателей

- Над каждым эвакуационным выходом
- В коридорах — через каждые 25 м (чтобы из любой точки был виден хотя бы один)
- На поворотах и пересечениях
- На лестничных клетках — на каждой площадке
- У средств пожаротушения и оповещения

## 5. Кабельные линии аварийного освещения

### Требования к прокладке

| Требование | Описание |
|---|---|
| Огнестойкость кабеля | Кабель с индексом FR (Fire Resistant) |
| Время работоспособности | ≥ 1 час при пожаре (для ЦБП) |
| Отдельная трасса | Кабели аварийного освещения отдельно от рабочего |
| Прокладка | В огнестойких коробах или замоноличенно |

### Марки кабелей

| Марка | Огнестойкость | Применение |
|---|---|---|
| ВВГнг(А)-FRLS | 180 мин при 750 °C | Стационарная прокладка в зданиях |
| КПСнг(А)-FRLS | 180 мин при 750 °C | Слаботочные линии (сигнализация) |

### Для автономных светильников

Огнестойкий кабель не требуется — светильник работает от встроенной батареи. Достаточно стандартного ВВГнг(А)-LS.

## 6. Проектирование — пошагово

### Шаг 1: Определить зоны

1. Отметить все пути эвакуации (коридоры, лестницы, тамбуры)
2. Определить зоны повышенной опасности (производство, кухни)
3. Отметить места средств пожаротушения
4. Определить эвакуационные выходы

### Шаг 2: Выбрать тип системы

```
Малый объект (до 50 точек) → Автономные светильники
Средний/большой объект → ЦБП + автономные указатели
Производство с высокими требованиями → ЦБП + ДГУ (дизель-генератор)
```

### Шаг 3: Разместить светильники

**Пути эвакуации:**
- Шаг ≤ 4× высота установки (от пола до светильника)
- На поворотах и пересечениях — обязательно
- У каждой двери на пути эвакуации
- У лестниц — так, чтобы каждая ступень была освещена

**Указатели:**
- Над выходами
- Через каждые 25 м на прямых участках
- На каждом повороте

### Шаг 4: Расчёт в DIALux

1. Создать отдельный расчёт аварийного освещения
2. Отключить рабочее освещение
3. Включить только аварийные светильники
4. Проверить: Ē ≥ 1 лк на полу путей эвакуации, U₀ ≥ 0,4

### Шаг 5: Документация

- План размещения аварийных светильников (отдельный чертёж или слой)
- Однолинейная схема ЩАО (щит аварийного освещения)
- Расчёт освещённости (DIALux)
- Спецификация оборудования
- Кабельный журнал (с указанием огнестойких кабелей)

## 7. Обслуживание и тестирование

### Регламент проверок (ГОСТ Р 55842)

| Проверка | Периодичность | Что проверять |
|---|---|---|
| Визуальный осмотр | Ежемесячно | Целостность, индикация заряда |
| Функциональный тест | Ежемесячно | Включение при отключении сети |
| Тест автономности (полный разряд) | Ежегодно | Время работы от батареи |
| Замена батарей | Каждые 3–5 лет | По результатам теста автономности |

### Автоматическое тестирование

Современные системы (DALI-совместимые) позволяют:
- Автоматический функциональный тест каждый месяц
- Автоматический тест автономности раз в год
- Удалённый мониторинг через BMS
- Журнал тестов для проверяющих органов

## 8. Типичные ошибки

| Ошибка | Последствия | Решение |
|---|---|---|
| Нет аварийного освещения | Не пройдёт экспертизу, штрафы | Проектировать с самого начала |
| Указатели не видны из любой точки | Паника при эвакуации | Шаг ≤ 25 м, на каждом повороте |
| Обычный кабель для ЦБП | Перегорит при пожаре | Использовать FR-кабель |
| Автономность < 1 часа | Нарушение норм | Проверять батареи ежегодно |
| Аварийное на одном автомате с рабочим | Отключение при перегрузке рабочего | Отдельный щит ЩАО |
| Нет тестирования | Батареи деградируют, система не работает | Регламент по ГОСТ Р 55842 |

## Итоги

- Аварийное освещение **обязательно** для всех общественных и производственных зданий
- **1 лк** на путях эвакуации, **≥ 1 час** автономной работы
- **Автономные светильники** — для малых объектов, **ЦБП** — для крупных
- Указатели «Выход» — над каждым выходом, видны из любой точки коридора
- Кабели для ЦБП — только **огнестойкие (FR)**
- **Ежегодное тестирование** автономности — обязательно
