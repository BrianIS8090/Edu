# Материаловедение для светотехники

Выбор материалов определяет внешний вид, надёжность, теплоотвод и стоимость светильника. Этот урок — справочник по материалам, которые используются при проектировании и производстве светотехники.

---

## 1. Металлы

### Алюминий и алюминиевые сплавы

Основной конструкционный материал для светильников благодаря лёгкости, теплопроводности и коррозионной стойкости.

| Сплав | Теплопроводность, Вт/(м·К) | Плотность, г/см³ | Применение |
|---|---|---|---|
| АД31 (6063) | 200 | 2,7 | Экструзионные профили, радиаторы |
| АМг2 (5052) | 138 | 2,68 | Листовые корпуса, отражатели |
| АК12 (A413) | 121 | 2,66 | Литые корпуса (литьё под давлением) |
| АМц (3003) | 193 | 2,73 | Тонколистовые детали |

**Экструзия vs Литьё vs Листовой металл:**

| Параметр | Экструзия | Литьё под давлением | Листовая штамповка |
|---|---|---|---|
| Форма | Линейный профиль | Сложная 3D-форма | Простые формы |
| Минимальная партия | 200–500 кг | 500–1000 шт. | 100+ шт. |
| Точность | ±0,2 мм | ±0,1 мм | ±0,3 мм |
| Теплоотвод | Отличный | Хороший | Средний |
| Стоимость оснастки | Низкая (матрица) | Высокая (пресс-форма) | Средняя (штамп) |
| Типичные изделия | Линейные светильники, радиаторы | Корпуса даунлайтов, прожекторов | Кронштейны, крышки |

### Сталь

Используется реже из-за веса и низкой теплопроводности, но необходима для:

- **Оцинкованная сталь** — монтажные пластины, крепёж, корпуса бюджетных светильников
- **Нержавеющая сталь (AISI 304/316)** — морские, пищевые, химические среды
- **Пружинная сталь** — клипсы для встраиваемых светильников

### Медь и латунь

- **Медь** — теплоотводы для мощных LED (теплопроводность 400 Вт/(м·К))
- **Латунь** — декоративные элементы, контактные группы

## 2. Полимеры (пластики)

### Поликарбонат (PC)

Основной материал для рассеивателей и защитных стёкол.

| Параметр | Значение |
|---|---|
| Светопропускание | 86–92% (прозрачный) |
| Ударная прочность | В 200 раз выше стекла |
| Рабочая температура | –40...+130 °C |
| УФ-стойкость | Требует стабилизатора |
| Горючесть | В1 / UL94 V-2 (самозатухающий) |

**Виды рассеивателей из PC:**

- **Прозрачный** — максимальный КПД, видны LED-точки
- **Матовый (опал)** — равномерное свечение, потери 10–15%
- **Микропризматический** — баланс КПД и равномерности, потери 5–8%

### Полиметилметакрилат — ПММА (оргстекло, акрил)

| Параметр | Значение |
|---|---|
| Светопропускание | 92% (лучше PC) |
| Ударная прочность | Ниже PC (в 10 раз выше стекла) |
| Рабочая температура | –40...+80 °C |
| УФ-стойкость | Отличная (не желтеет) |
| Горючесть | Горюч (не самозатухающий) |

**Когда ПММА лучше PC:**
- Декоративное освещение (прозрачность, блеск)
- Вывески и световые панели
- Низкотемпературные применения

**Когда PC лучше ПММА:**
- Наружное освещение (ударопрочность)
- Высокие температуры (близко к LED)
- Требования по пожарной безопасности

### Другие полимеры

| Полимер | Применение | Особенности |
|---|---|---|
| PA (нейлон) | Клеммные колодки, разъёмы | Электроизоляция, термостойкость |
| PBT | Цоколи, патроны | Стойкость к нагреву, не горит |
| ABS | Декоративные корпуса | Легко красится, дёшев |
| Силикон | Герметизация, линзы, прокладки | Эластичность –60...+200 °C |
| POM (делрин) | Крепёжные элементы, защёлки | Прочность, пружинные свойства |

## 3. Оптические материалы

### Стекло

| Тип | Светопропускание | Применение |
|---|---|---|
| Закалённое | 90% | Наружные светильники, прожекторы |
| Боросиликатное | 91% | Высокотемпературные применения |
| Матовое (пескоструй) | 75–85% | Рассеиватели |
| Закалённое защитное | 88% | Грунтовые светильники, IP67+ |

### Линзы

| Материал | Применение | Рабочая температура |
|---|---|---|
| ПММА | Вторичная оптика (линзы, рефлекторы) | до 80 °C |
| PC | Линзы для мощных LED | до 130 °C |
| Силикон | Первичная оптика (на LED-чипе) | до 200 °C |
| Стекло | Прецизионная оптика | до 500 °C |

### Отражатели

| Материал | Коэффициент отражения | Применение |
|---|---|---|
| Анодированный алюминий (MIRO) | 95–98% | Высококачественные отражатели |
| Полированный алюминий | 85–90% | Бюджетные рефлекторы |
| Белое покрытие | 90–95% | Диффузное отражение |
| Вакуумная металлизация | 85–92% | Пластиковые отражатели |

## 4. Покрытия и обработка поверхности

### Анодирование алюминия

Электрохимическое создание оксидной плёнки на поверхности.

| Параметр | Стандартное | Твёрдое |
|---|---|---|
| Толщина плёнки | 10–25 мкм | 40–80 мкм |
| Твёрдость | 200–400 HV | 400–600 HV |
| Коррозионная стойкость | Хорошая | Отличная |
| Цвет | Натуральный, чёрный, цветной | Тёмно-серый, чёрный |
| Стоимость | Низкая | Высокая |
| Применение | Декоративные детали | Наружные светильники |

### Порошковая окраска

Нанесение полимерного порошка с последующим запеканием при 180–200 °C.

| Параметр | Значение |
|---|---|
| Толщина покрытия | 60–120 мкм |
| Адгезия | Отличная (при правильной подготовке) |
| Стойкость к царапинам | Хорошая |
| Палитра | RAL, NCS, любой цвет |
| Подготовка поверхности | Обезжиривание → фосфатирование |

**Типы порошковых красок:**

| Тип | Стойкость к УФ | Применение |
|---|---|---|
| Полиэфирная | Отличная (до 10 лет на улице) | Наружные светильники |
| Эпоксидная | Плохая (желтеет) | Только интерьер |
| Эпоксидно-полиэфирная | Средняя | Интерьер, навесы |

### Другие покрытия

| Покрытие | Назначение |
|---|---|
| Гальваническое хромирование | Декоративный блеск |
| Цинкование (горячее / гальваническое) | Антикоррозийная защита стали |
| Вакуумная металлизация | Зеркальный эффект на пластике |
| Лакирование | Защита анодированного слоя |

## 6. Критерии выбора материалов

### Чек-лист для проектировщика

1. **Среда эксплуатации**
   - Интерьер → больше вариантов, дешевле
   - Экстерьер → УФ-стойкость, коррозионная стойкость
   - Агрессивная среда → нержавейка, специальные покрытия

2. **Температурный режим**
   - Мощность LED → нагрев → выбор теплоотвода и рассеивателя
   - Температура эксплуатации → рабочий диапазон материала

3. **Пожарная безопасность**
   - Класс горючести по ГОСТ 30244
   - UL94 для полимеров
   - Не использовать горючие материалы в путях эвакуации

4. **Оптические требования**
   - КПД светильника → светопропускание рассеивателя
   - Распределение света → тип оптики и отражателя
   - Блёсткость → матирование, призма

5. **Стоимость и технологичность**
   - Объём производства → выбор технологии (экструзия / литьё / штамповка)
   - Доступность материала
   - Стоимость обработки и покрытия

## Итоги

- **Алюминий** — основа конструкции: экструзия для линейных, литьё для сложных форм
- **Поликарбонат** — рассеиватели для наружки и мощных светильников (ударопрочность)
- **ПММА** — рассеиватели для интерьера (прозрачность, не желтеет)
- **Порошковая окраска** — полиэфир для улицы, эпоксидная только для интерьера
- **Анодирование** — защита + декор для алюминиевых деталей
- Выбор материала — всегда компромисс между оптикой, теплом, защитой и ценой
