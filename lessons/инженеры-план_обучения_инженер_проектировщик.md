# Памятка для инженера-проектировщика

***

## Этап 1. Организационный

**Цель:** настроить рабочее окружение и получить доступы.

### 1. Корпоративная почта и мессенджер

* Получить доступ к корпоративной почте (mail.ru) - выдается работодателем.

* Установить VKteams и зарегистрироваться через корпоративную почту  (десктоп + телефон), настроить аватар

### 2. Система управления проектами

* Зарегистрироваться в [WEEEK.net](https://app.weeek.net/) через корпоративную почту (обязательное указание реального имени и фамилии)

* Привязать Telegram-бота для уведомлений WEEEK

* Установить приложение на ПК и телефон, настроить аватар

### 3. ERP-система компании

* Перейти на корпоративная ERP-система для управления складом, проектами и спецификациями

* Зарегистрироваться через корпоративную почту (доступ предоставляется администратором)

* Ознакомиться с основными модулями: Склад, Проекты, Спецификации

### 4. Презентации и обсуждение проектов

* Зарегистрироваться на [holst.so](https://holst.so/) (корп. почта)

* **Правило:** любая ссылка на интернет-ресурс дублируется скриншотом

### 5. Дополнительное ПО

* WireGuard (VPN, ключ по запросу)

* OnlyOffice — документы/таблицы

* PDFsam — работа с PDF

* Настроить [шорткаты и скрипты](https://app.weeek.net/ws/715837/kb/60)

## Этап 3. Оформление документации

**Цель:** научиться оформлять чертежи и документацию по регламентам.

### 1. Rhino3D v8.22 (оформление документации)

* Установить с [Яндекс.Диска](https://disk.yandex.ru/d/V9chqUq-RxwUjw)

* Загрузить [готовые настройки](https://app.weeek.net/ws/715837/kb/89)

* Установить [шрифт Iosevka Fixed](https://app.weeek.net/ws/715837/kb/59)

* Изучить [скрипты Rhino](https://app.weeek.net/ws/715837/kb/52)

* Настроить [прямое копирование из Rhino в Moi](https://app.weeek.net/ws/715837/kb/43)

* Подключить библиотеку 3D-моделей: `O:\Проектный отдел\Библиотеки\Библиотека 3D-моделей.3dm`

### 2. Регламенты

* Изучить [регламенты по оформлению чертежей](https://app.weeek.net/ws/715837/kb/88)

* Изучить [оформление ТЗ на лазерную резку и гибку](https://app.weeek.net/ws/715837/kb/51)

### 3. CorelDraw 2021 (паспорта и наклейки)

* Установить с [Яндекс.Диска](https://disk.yandex.ru/d/Q_OJCmnkui1NJw) + [обновление](https://disk.yandex.ru/d/Hgk_Q7U5Yg_Rbg)

## Этап 4. Светотехника

**Цель:** освоить основы светотехники и инструменты расчёта.

### 1. Теоретическая база

* Изучить [основы светотехники для инженера-проектировщика](https://app.weeek.net/ws/715837/kb/87)

### 2. DIALux Evo 13.2 (светотехнические расчёты)

* Установить с [официального сайта](https://www.dialux.com/en-GB/download)

### 3. Поставщики — ознакомиться с основными

* [E-neon](https://e-neon.ru/) — LED-источники, блоки питания

* [Svetolog](https://svetolog.ru/) — светодиодная лента, БП 12–24V

* [Transistor](https://transistor.ru/) — управление светом

***

## Этап 5. Производственные технологии

**Цель:** понимать технологические ограничения производства.

### 1. Металлообработка

* Изучить [теорию по гибке металла](https://app.weeek.net/ws/715837/kb/67)

* Ознакомиться с поставщиками алюминиевых корпусов: [Teclog](https://teclog.ru/led-lamp), [LEDCraft](https://ledcraft.ru/)

### 2. 3D-печать

* Установить слайсер: [OrcaSlicer](https://github.com/SoftFever/OrcaSlicer) или [CrealityPrint](https://github.com/CrealityOfficial/CrealityPrint)

* Для фотополимерной печати: [PhotonWorkshop](https://github.com/ANYCUBIC-3D/PhotonWorkshop), [Chitubox](https://www.chitubox.com/)

* Ознакомиться с поставщиками пластика: [REC3D](https://rec3d.ru/), [Sopytka](https://www.sopytka.ru/)

* Литьевой пластик и силикон: [Caspol](https://caspol.ru/)

### 3. Реверс-инжиниринг

* Установить [QuickSurface 2026](https://disk.yandex.ru/d/zcfd0UR1nGfXUw)

* Изучить [обучающие материалы QuickSurface](https://app.weeek.net/ws/715837/kb/90)

* Установить ПО для сканера [Creality CR-Scan Raptor PRO](https://www.crealitycloud.com/downloads/software/creality-scan/Windows)
