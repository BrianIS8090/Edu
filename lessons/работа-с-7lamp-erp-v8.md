# Работа с 7LAMP ERP v8.0

Руководство по работе с системой управления предприятием 7LAMP ERP версии 8. Вы узнаете, как войти в систему, ориентироваться в интерфейсе и использовать основные модули: Склад, Проекты и Спецификации.

---

## 1. Вход в систему

Откройте браузер и перейдите по адресу, который вам предоставил администратор. Появится страница авторизации.

**Для входа нужны:**
- Email — корпоративная почта, зарегистрированная в системе
- Пароль — выдаётся администратором при создании учётной записи

После ввода данных нажмите **Войти**. Система перенаправит вас на главную страницу — модуль «Склад».

> Если вы забыли пароль — обратитесь к администратору. Самостоятельный сброс пароля недоступен.

---

## 2. Интерфейс и навигация

### Верхняя панель (Header)

В верхней части экрана расположена панель навигации:

| Элемент | Описание |
|---------|----------|
| Логотип **7LAMP ERP v8.0** | Возврат на главную |
| Вкладка **Склад** | Модуль управления складом |
| Вкладка **Проекты** | Модуль управления проектами |
| Вкладка **Спецификации** | Модуль спецификаций |
| Иконка солнце/луна | Переключение светлой и тёмной темы |
| Аватар пользователя | Меню профиля и выход |

Активная вкладка подсвечивается синей линией снизу. На мобильных устройствах навигация скрывается в меню-бургер (три полоски).

### Переключение темы

Нажмите на иконку солнца (светлая тема) или луны (тёмная тема) в верхней панели. Выбор сохраняется автоматически — при следующем входе тема не сбросится.

### Меню профиля

Нажмите на свой аватар в правом верхнем углу. Откроется выпадающее меню:

- **Мой профиль** — просмотр и редактирование личных данных
- **Админ-панель** — доступна только администраторам
- **Выход** — завершение сессии

## 3. Модуль «Склад»

Основной модуль системы. Здесь ведётся учёт всех товаров и материалов компании.

![Модуль «Склад»](assets/работа-с-7lamp-erp-v8/склад.png)

### 3.1. Просмотр товаров

Товары отображаются в виде **таблицы** или **карточек**. Переключение режима — кнопки в верхней части модуля.

**Столбцы таблицы:**

| Столбец | Описание |
|---------|----------|
| Название | Наименование товара |
| Производитель | Компания-изготовитель |
| Категория | Группа товара |
| Ед. изм. | Единица измерения (шт, м, кг и т.д.) |
| Количество | Текущий остаток на складе |
| Свободно | Количество минус резерв |
| Стоимость | Цена за единицу |
| Сумма | Количество × Стоимость |

Для сортировки нажмите на заголовок столбца. Повторное нажатие меняет направление сортировки.

### 3.2. Поиск и фильтрация

**Поиск** — введите текст в строку поиска. Система ищет по названию и производителю.

**Фильтры категорий** — нажмите на кнопку категории для фильтрации по ней.

**Быстрые фильтры** — специальные кнопки для быстрого отбора:

| Фильтр | Что показывает |
|--------|---------------|
| Нулевой остаток | Товары с количеством = 0 |
| Низкий остаток | Товары с минимальным количеством |
| Отрицательные | Товары с отрицательным остатком (ошибки учёта) |
| Зарезервировано | Товары, которые зарезервированы под проекты |
| Нет фото | Товары без загруженного изображения |
| Устаревшая цена | Товары, у которых давно не обновлялась стоимость |

### 3.3. Добавление товара

1. Нажмите кнопку **Добавить товар**
2. Заполните обязательные поля:
   - **Название** — наименование товара
   - **Производитель** — компания-изготовитель
   - **Категория** — выберите из списка или введите новую
   - **Единица измерения** — шт, м, кг, компл и т.д.
   - **Количество** — начальный остаток
   - **Стоимость** — цена за единицу
3. Опционально загрузите фото и укажите ссылку на поставщика
4. Нажмите **Сохранить**

### 3.4. Редактирование товара

Дважды кликните на строку товара в таблице — откроется карточка редактирования. Измените нужные поля и сохраните.

### 3.5. Приход и списание товаров

Для изменения остатка на складе используйте операции **движения**:

**Приход (поступление):**
1. Выберите товар
2. Нажмите кнопку прихода (стрелка вниз)
3. Укажите количество и номер накладной (если есть)
4. Подтвердите операцию

**Списание (расход):**
1. Выберите товар
2. Нажмите кнопку списания (стрелка вверх)
3. Укажите количество и причину
4. Подтвердите операцию

> Все движения фиксируются в истории и отображаются в карточке товара.

### 3.6. Массовые операции

- **Выделение нескольких товаров** — используйте чекбоксы слева от строк
- **Массовое удаление** — выделите товары и нажмите кнопку удаления
- **Экспорт в Excel** — нажмите иконку таблицы для выгрузки данных

### 3.7. Статистика склада

Нажмите кнопку статистики для просмотра:
- Общая стоимость склада
- Количество позиций
- Топ товаров по стоимости
- Распределение по категориям (диаграмма)

### 3.8. Поиск дубликатов

Система умеет находить товары с похожими названиями. Нажмите кнопку «Дубликаты» для анализа и объединения позиций.

## 4. Модуль «Проекты»

Управление проектами компании: отслеживание статусов, дедлайнов и назначенных сотрудников.

![Модуль «Проекты»](assets/работа-с-7lamp-erp-v8/проекты.png)

### 4.1. Просмотр проектов

Проекты отображаются в виде таблицы или карточек. Каждый проект содержит:
- Название и описание
- Статус (активный, завершён, приостановлен)
- Дедлайн
- Назначенные сотрудники
- Связанные спецификации

### 4.2. Создание проекта

1. Нажмите **Создать проект**
2. Заполните название, описание и дедлайн
3. Назначьте ответственных
4. Сохраните

### 4.3. Статистика

В модуле доступна сводная статистика: количество активных проектов, просроченные дедлайны, загрузка сотрудников.

---

## 5. Модуль «Спецификации»

Модуль для работы со спецификациями изделий — перечнями компонентов и материалов.

![Модуль «Спецификации»](assets/работа-с-7lamp-erp-v8/спеицификации.png)

### 5.1. Просмотр спецификации

Откройте нужную спецификацию из списка. Отображается таблица с компонентами:
- Название компонента
- Количество
- Единица измерения
- Стоимость

### 5.2. Редактирование

Откройте спецификацию и внесите изменения. Данные сохраняются автоматически через несколько секунд после изменения.

### 5.3. Копирование и печать

- **Копирование** — создаёт дубликат спецификации для нового проекта
- **Печать** — формирует версию для вывода на принтер

## 6. Профиль пользователя

Перейдите в **Мой профиль** через меню в правом верхнем углу.

**Доступно:**
- Просмотр и редактирование имени, аватара
- Просмотр своей роли и прав доступа
- История активности (последние действия в системе)

---

## 7. Роли и права доступа

Каждому пользователю назначается роль, определяющая доступные действия.

| Роль | Описание | Основные права |
|------|----------|---------------|
| Администратор | Полный доступ ко всем функциям | Всё + управление пользователями |
| Директор | Руководство компанией | Просмотр и редактирование всех модулей |
| Коммерческий директор | Управление продажами | Проекты, коммерческий отдел |
| Менеджер продаж | Работа с клиентами | Проекты, каталог |
| Инженер | Проектирование | Спецификации, каталог, склад (просмотр) |
| Начальник производства | Управление производством | Склад, спецификации |
| Начальник склада | Управление складом | Полный доступ к складу, движения |
| Гость | Только просмотр | Просмотр без изменений |

> Если вам не хватает прав для выполнения задачи — обратитесь к администратору.

## 8. Частые вопросы

**Не могу войти в систему**
Проверьте правильность email и пароля. Если не помогает — обратитесь к администратору для сброса пароля.

**Не вижу кнопку добавления/удаления**
Ваша роль не имеет прав на эту операцию. Обратитесь к администратору.

**Данные не обновляются**
Система работает в реальном времени. Если данные не появляются — проверьте подключение к интернету и обновите страницу (F5).

**Как изменить тему оформления?**
Нажмите иконку солнца/луны в верхней панели. Выбор сохраняется автоматически.

**Как выгрузить данные в Excel?**
В модуле «Склад» нажмите иконку таблицы (экспорт) в панели инструментов.
