# Драйверы и блоки питания LED

Драйвер — «сердце» светильника. Он преобразует сетевое напряжение в стабильный ток или напряжение для LED. Неправильный выбор драйвера — мерцание, перегрев, преждевременный выход из строя. Этот урок — полное руководство по типам, выбору и подключению драйверов.

---

## 1. Токовый vs Напряженческий

### Токовый драйвер (Constant Current, CC)

Поддерживает постоянный ток на выходе независимо от нагрузки.

| Параметр | Типичные значения |
|---|---|
| Выходной ток | 150 мА, 250 мА, 350 мА, 500 мА, 700 мА, 1050 мА |
| Диапазон напряжений | Плавающий (например, 20–42 В) |
| Нагрузка | LED-модули последовательно |

**Когда использовать:**
- Дискретные LED-модули (COB, SMD-платы)
- Даунлайты, трековые, прожекторы
- Критична стабильность светового потока

**Схема подключения:**
```
AC 230V → [CC Driver 700mA, 20-42V] → LED1 → LED2 → LED3 (последовательно)
```

### Напряженческий блок питания (Constant Voltage, CV)

Поддерживает постоянное напряжение на выходе.

| Параметр | Типичные значения |
|---|---|
| Выходное напряжение | 12 В DC или 24 В DC (реже 48 В) |
| Максимальный ток | Зависит от мощности (например, 150 Вт / 24 В = 6,25 А) |
| Нагрузка | LED-ленты, модули с встроенными резисторами |

**Когда использовать:**
- LED-ленты
- Вывески
- Системы с параллельным подключением модулей
- Длинные линии освещения

**Схема подключения:**
```
AC 230V → [CV Driver 24V, 150W] → LED-лента (параллельно, до макс. тока)
```

### Сравнение

| Критерий | Токовый (CC) | Напряженческий (CV) |
|---|---|---|
| Стабильность яркости | Отличная | Зависит от качества ленты |
| Простота подключения | Последовательно | Параллельно |
| Гибкость по нагрузке | Только определённые модули | Любая нагрузка до макс. мощности |
| Риск перегрузки | Низкий (ток ограничен) | Возможен при КЗ ленты |
| Типичное применение | Светильники | LED-ленты, вывески |

## 2. Ключевые параметры драйверов

### Электрические параметры

| Параметр | Описание | На что влияет |
|---|---|---|
| Мощность (P) | Максимальная нагрузка, Вт | Запас ≥ 10–20% от потребления LED |
| КПД (η) | Эффективность, % | Потери = нагрев; хорошие драйверы: η > 90% |
| Коэффициент мощности (PF) | Нагрузка на сеть | PF > 0,9 для коммерческих объектов |
| THD | Гармонические искажения, % | THD < 20% по ГОСТ |
| Пусковой ток (Inrush) | Пиковый ток при включении | Определяет макс. кол-во на автомат |

### Параметры безопасности

| Параметр | Описание |
|---|---|
| Класс изоляции | I (с заземлением) или II (двойная изоляция) |
| IP-защита | IP20 (сухие помещения), IP65–IP67 (влага/улица) |
| Рабочая температура (Tc) | Температура корпуса в точке Tc (обычно 75–90 °C) |
| Срок службы | Часы до отказа при заданной Tc (обычно 50 000–100 000 ч) |

### Параметры диммирования

| Протокол | Описание | Провода |
|---|---|---|
| DALI / DALI-2 | Цифровой, 64 адреса, сцены | + 2 провода (DA+, DA–) |
| 0-10V / 1-10V | Аналоговый, простой | + 2 провода (DIM+, DIM–) |
| Push-Dim | Кнопочное управление | + 1 провод (через фазу) |
| Phase-Cut (TRIAC) | Через настенный диммер | Без дополнительных проводов |
| Casambi / Bluetooth | Беспроводное | Нет проводов управления |
| DALI-2 DT8 (Tunable White) | Управление CCT | + 2 провода DA |

## 3. Выбор драйвера — пошаговый алгоритм

### Шаг 1: Определить тип LED-нагрузки

```
LED-лента 24 В → Напряженческий (CV) 24 В
LED-модуль COB 35 В / 700 мА → Токовый (CC) 700 мА
```

### Шаг 2: Рассчитать мощность

```
Для CC: P = U_LED × I_LED × кол-во последовательных LED
         + запас 15–20%

Для CV: P = P_LED_на_метр × длина_ленты
         + запас 15–20%
```

**Пример (CC):** 3 модуля COB × 35 В × 0,7 А = 73,5 Вт → драйвер на 100 Вт

**Пример (CV):** Лента 14,4 Вт/м × 8 м = 115,2 Вт → блок питания на 150 Вт

### Шаг 3: Проверить диапазон напряжений (для CC)

Напряжение на LED-модулях не должно выходить за рабочий диапазон драйвера:

```
Драйвер: 700 мА, 20–42 В
LED-модули: 3 × 12 В = 36 В → попадает в 20–42 В ✓
```

### Шаг 4: Выбрать тип диммирования

| Задача | Протокол | Драйвер |
|---|---|---|
| Простое вкл/выкл | — | Без диммирования |
| Настенный диммер | Phase-Cut | TRIAC-совместимый |
| Управление со сцены | DALI | DALI-совместимый |
| Беспроводное | Casambi | С модулем Casambi |
| Tunable White | DALI DT8 | Двухканальный DALI DT8 |

### Шаг 5: Определить условия эксплуатации

```
Сухое помещение → IP20, класс II
Ванная / навес → IP44–IP65
Улица / грунт → IP65–IP67
```

### Шаг 6: Проверить температурный режим

Срок службы драйвера прямо зависит от температуры в точке Tc:

| Tc, °C | Типичный срок службы |
|---|---|
| 60 | 100 000 ч |
| 70 | 70 000 ч |
| 80 | 50 000 ч |
| 90 | 30 000 ч |

**Правило:** Tc_рабочая < Tc_макс на 10–15 °C для запаса.

## 5. Схемы подключения

### Один драйвер — один светильник

```
L ──┐
N ──┤ [CC Driver] ──→ LED-модуль
PE ─┘
```

Самая надёжная схема. Отказ драйвера = выход из строя одного светильника.

### Один драйвер — несколько светильников (CC)

```
L ──┐
N ──┤ [CC Driver 700mA] ──→ LED₁ ──→ LED₂ ──→ LED₃
PE ─┘                      (последовательно)
```

**Ограничения:**
- Суммарное напряжение LED ≤ макс. напряжение драйвера
- Отказ одного LED = отключение всей цепочки

### Один блок питания — несколько лент (CV)

```
L ──┐                      ┌→ Лента 1 (5 м)
N ──┤ [CV Driver 24V] ─────┤→ Лента 2 (5 м)
PE ─┘                      └→ Лента 3 (5 м)
                           (параллельно)
```

**Ограничения:**
- Суммарный ток ≤ макс. ток драйвера
- Длина одной ветви ленты ≤ 5 м (падение напряжения)
- Для длинных линий — подвод питания с двух сторон или промежуточное питание

## 6. Пусковой ток и автоматы

### Проблема

LED-драйверы имеют конденсаторы на входе, которые при включении потребляют огромный кратковременный ток:

```
Рабочий ток HLG-150H: 0,7 А
Пусковой ток: 65 А × 530 мкс
```

### Расчёт количества драйверов на автомат

Производители указывают максимальное количество драйверов на автомат:

| Драйвер | Автомат B16 | Автомат C16 |
|---|---|---|
| Mean Well HLG-150H | 8 шт. | 12 шт. |
| Mean Well HLG-240H | 6 шт. | 9 шт. |
| Mean Well LCM-60 | 12 шт. | 18 шт. |

**Важно:** Автомат типа B срабатывает быстрее → меньше драйверов на линию. Тип C — предпочтительнее для LED-освещения.

### Решения при большом количестве драйверов

- Автоматы типа C или D вместо B
- Ограничители пускового тока (NTC-термисторы, электронные ограничители)
- Поочерёдное включение групп (soft start через контакторы с задержкой)

## 7. Типичные ошибки

| Ошибка | Последствия | Решение |
|---|---|---|
| Драйвер без запаса мощности | Перегрев, быстрый износ | Запас 15–20% по мощности |
| CC-драйвер для LED-ленты | Нестабильная яркость, повреждение | Использовать CV для лент |
| Не учтён пусковой ток | Срабатывание автомата при включении | Проверить таблицу inrush |
| TRIAC-диммер с не-TRIAC драйвером | Мерцание, гудение | Проверить совместимость |
| Tc выше допустимой | Срок службы падает в 2–3 раза | Обеспечить вентиляцию |
| Драйвер IP20 на улице | Выход из строя | IP65+ для наружного применения |

## Итоги

- **Токовый (CC)** — для LED-модулей, последовательное подключение
- **Напряженческий (CV)** — для LED-лент, параллельное подключение
- Запас мощности ≥ 15%, температура Tc < максимума на 10 °C
- **DALI** — стандарт для коммерческих объектов; **0-10V** — для простых задач
- **Пусковой ток** — главный фактор при выборе автоматов
- Mean Well — самый доступный и документированный бренд
