# Инструменты CAD: Моделирование в Moi3D

В арсенале инженера-проектировщика светотехники **Moi3D** применяется исключительно для прямого 3D-моделирования геометрии. 

Ниже мы подробно рассмотрим, какие именно инструменты предоставляет эта программа для решения инженерных задач.

---

## Moi3D: Инструменты для 3D-моделирования

Moi3D оптимизирован для быстрого твердотельного NURBS-моделирования. Весь интерфейс построен вокруг быстрого доступа к командам создания и редактирования геометрии корпусов, линз, плат и профилей.

### Создание базовой геометрии (Draw Curve / Solid)
Всё начинается с профилей и примитивов:
* **Line, Circle, Rect** — создание плоских эскизов для профилей экструзии.
* **Box, Cylinder, Sphere** — создание базовых твердотельных примитивов, из которых "вырезается" форма светильника.

### Инструменты формообразования (Construct)
Группа инструментов, превращающих плоские эскизы в 3D-объекты:
* **Extrude (Выдавливание)** — основной инструмент для линейных светильников и алюминиевых профилей.
* **Revolve (Вращение)** — создание корпусов даунлайтов, оптических линз и круглых рефлекторов вокруг оси.
* **Sweep (Сдвиг) / Loft (Лофт)** — построение сложных переходных форм (например, плафонов сложной кривизны) по направляющим кривым.
* **Offset / Shell (Оболочка)** — мгновенное создание пустотелого корпуса из сплошного объёма с заданной толщиной стенки.

### Булевы операции (Boolean)
Важнейший блок для конструктора — добавление и вычитание объемов:
* **Difference (Вычитание)** — вырезание отверстий под крепеж, вентиляционных пазов, посадочных мест под LED-платы.
* **Union (Объединение)** — слияние нескольких пересекающихся деталей (профиля и заглушек) в единый литой корпус.
* **Intersection (Пересечение)** — получение детали из зоны пересечения двух объемов.

### Инструменты сопряжения (Fillet / Chamfer)
* **Fillet (Скругление)** — сглаживание острых углов для литейных форм или фрезеровки (расчёт под радиус фрезы).
* **Chamfer (Фаска)** — снятие технологических фасок на кромках деталей для сборки без заусенцев.

*Ключевая задача в Moi3D — подготовить чистую 3D-модель твердого тела (Closed Solid) для передачи на производство (STEP) или для импорта в Rhino для создания чертежей.*
