# 3D-моделирование и CAD для инженера

Инженер-проектировщик светильников работает с 3D-моделями на всех этапах — от концепта до передачи на производство. Этот урок покрывает практические приёмы работы в Moi3D и Rhino, подготовку моделей к производству и типичные ошибки.

---

## 1. Инструменты проектировщика

### Moi3D v5

Основной инструмент для быстрого 3D-моделирования. Интуитивный интерфейс, NURBS-моделирование.

**Сильные стороны:**
- Быстрое создание корпусов и профилей
- Отличный экспорт в STEP/IGES для производства
- Булевы операции без артефактов
- Простой интерфейс — минимальное время обучения

**Горячие клавиши (must know):**

| Клавиша | Действие |
|---|---|
| Tab | Переключение между видами |
| S | Привязки (snap) |
| D | Прямое редактирование |
| Q | Быстрый выбор |
| F | Фаска (Fillet) |
| Ctrl+Shift+S | Сохранить как |
| Space | Повтор последней команды |

### Rhino 8

Для сложных задач, визуализации и работы с Grasshopper (параметрическое проектирование).

**Когда Rhino вместо Moi3D:**
- Параметрические модели (Grasshopper)
- Сложные поверхности (органические формы)
- Визуализация и рендеринг
- Анализ модели (кривизна, толщина стенок)
- Работа с облаками точек (реверс-инжиниринг)

## 2. Моделирование корпуса светильника

### Типичный workflow

```
Эскиз/ТЗ → 2D-профиль → 3D-тело → Сопряжения → Отверстия → Проверка → Экспорт
```

### Шаг 1: Начинаем с профиля

Для линейных светильников (экструзия):

1. Нарисовать сечение профиля в плоскости XZ
2. Учесть толщину стенок (минимум 1,5 мм для алюминия)
3. Добавить элементы крепления LED-платы и рассеивателя
4. Предусмотреть пазы для торцевых заглушек

**Контрольные размеры профиля:**

| Элемент | Мин. размер |
|---|---|
| Толщина стенки (алюминий) | 1,2 мм (экструзия), 1,5 мм (литьё) |
| Паз для рассеивателя | Ширина рассеивателя + 0,2 мм зазор |
| Площадка для LED-платы | Ширина платы + 0,5 мм |
| Радиус скругления | ≥ 0,5 мм (наружный), ≥ 0,3 мм (внутренний) |

### Шаг 2: Экструзия / Булевы операции

- **Экструзия (Extrude)** — для линейных профилей
- **Вращение (Revolve)** — для круглых корпусов (даунлайты)
- **Булевы (Boolean)** — вычитание отверстий, пазов
- **Оболочка (Shell)** — создание полости внутри тела

### Шаг 3: Сопряжения и скругления

**Порядок скруглений (Fillet):**
1. Сначала — большие радиусы
2. Потом — маленькие
3. В последнюю очередь — фаски на острых кромках

**Радиусы для производства:**

| Технология | Минимальный внутренний радиус |
|---|---|
| Фрезеровка | R = диаметр фрезы / 2 (обычно R1–R3) |
| Литьё | R ≥ 0,5 мм |
| Гибка листа | R ≥ толщина листа |
| Экструзия | R ≥ 0,3 мм |

## 3. Подготовка к производству

### Развёртки (для листового металла)

При проектировании деталей из листа нужно учитывать:

1. **Радиус гиба** — зависит от толщины и материала
2. **К-фактор** — коэффициент положения нейтральной линии (обычно 0,3–0,5)
3. **Минимальная полка** — расстояние от гиба до края ≥ 2× толщина
4. **Направление проката** — гиб перпендикулярно прокату надёжнее

**Таблица минимальных радиусов гиба:**

| Материал | Толщина, мм | Мин. радиус гиба |
|---|---|---|
| Алюминий АМг2 | 1,0 | 1,0 мм |
| Алюминий АМг2 | 1,5 | 1,5 мм |
| Алюминий АМг2 | 2,0 | 2,0 мм |
| Сталь 08пс | 1,0 | 0,5 мм |
| Сталь 08пс | 1,5 | 1,0 мм |

### Допуски и посадки

| Класс точности | Допуск | Где применяется |
|---|---|---|
| Грубый (IT14) | ±0,5 мм | Общие габариты, кронштейны |
| Средний (IT12) | ±0,2 мм | Отверстия под крепёж, пазы |
| Точный (IT9–IT10) | ±0,05 мм | Посадочные места оптики, LED |
| Высокоточный (IT7) | ±0,02 мм | Прецизионная оптика |

**Правила простановки допусков:**

- Указывать допуски только на функциональные размеры
- Остальные размеры — по общему допуску (в основной надписи: «Неуказанные предельные отклонения по ГОСТ 30893.1-m»)
- Для отверстий под саморезы: ∅ отверстия = ∅ самореза + 0,3 мм

### Экспорт для производства

| Формат | Назначение |
|---|---|
| STEP (.stp) | Универсальный для ЧПУ, литья, 3D-печати |
| DXF (.dxf) | Развёртки для лазерной резки |
| STL (.stl) | 3D-печать прототипов |
| IGES (.igs) | Обмен между САПР |
| PDF с чертежом | Для цеха (размеры, допуски, покрытие) |

## 4. Параметрическое моделирование (Grasshopper)

### Зачем параметрика в светотехнике

- **Линейка светильников** — одна модель, разные длины (600, 900, 1200, 1500 мм)
- **Вариации крепления** — накладной / подвесной / встраиваемый
- **Оптимизация радиатора** — перебор параметров рёбер
- **Перфорация** — паттерны отверстий по алгоритму

### Базовый пример: параметрический линейный светильник

**Входные параметры:**
- Длина (slider: 300–3000 мм)
- Ширина профиля (slider: 30–100 мм)
- Высота профиля (slider: 20–80 мм)
- Толщина стенки (slider: 1,2–3,0 мм)

**Выходы:**
- 3D-модель корпуса
- Развёртка торцевых заглушек
- Длина LED-ленты
- Площадь поверхности (для расчёта теплоотдачи)

## 5. Моделирование под конкретные технологии

### Для экструзии алюминия

- Все элементы профиля — в одной плоскости сечения
- Нет поднутрений (undercuts)
- Толщина стенок ≥ 1,2 мм
- Соотношение max/min толщины стенок ≤ 3:1
- Симметрия — по возможности (равномерный поток металла)

### Для литья под давлением (алюминий)

- Уклоны ≥ 1° на наружных поверхностях
- Уклоны ≥ 2° на внутренних поверхностях
- Равномерная толщина стенок (избежать усадочных раковин)
- Рёбра жёсткости: толщина = 0,6× толщина стенки
- Радиусы на всех углах (облегчение извлечения из формы)

### Для 3D-печати (прототипы)

- FDM (пластик): точность ±0,2 мм, минимальная стенка 1,0 мм
- SLA (фотополимер): точность ±0,1 мм, минимальная стенка 0,5 мм
- SLS (нейлон): точность ±0,15 мм, минимальная стенка 0,8 мм

**Для прототипов светильников:** SLA для проверки формы, FDM для функциональных тестов (крепление, сборка).

## 6. Чек-лист перед передачей в производство

- [ ] Толщины стенок ≥ минимума для выбранной технологии
- [ ] Радиусы скругления на всех кромках
- [ ] Уклоны (для литья)
- [ ] Допуски на функциональные размеры
- [ ] Посадочные места LED-плат и драйверов проверены по даташитам
- [ ] Отверстия под крепёж соответствуют стандартным саморезам/винтам
- [ ] Развёртка разворачивается без наложений (для листового металла)
- [ ] STEP-файл экспортирован, открывается в другой САПР без ошибок
- [ ] Чертёж с размерами, допусками и указанием покрытия
- [ ] Согласование с технологом производства

## 7. Типичные ошибки

| Ошибка | Последствия | Решение |
|---|---|---|
| Нет зазора между деталями | Не собирается | Зазор 0,1–0,3 мм на сторону |
| Острые внутренние углы | Фреза не пройдёт, концентратор напряжений | Радиус ≥ R1 |
| Тонкие стенки рядом с отверстиями | Разрушение при сборке | Расстояние от отверстия до края ≥ 2× диаметр |
| Не учтён радиус гиба в развёртке | Деталь не совпадает по размерам | Использовать K-фактор |
| STEP с незашитыми поверхностями | Ошибки на ЧПУ | Проверить водонепроницаемость (closed solid) |

## Итоги

- **Moi3D** — для быстрого моделирования корпусов; **Rhino + Grasshopper** — для параметрики и сложных форм
- Всегда проектировать с учётом технологии: экструзия, литьё, листовой металл
- Допуски — только на функциональные размеры, остальное по общему допуску
- STEP — основной формат обмена, DXF — для лазерной резки
- Прототипирование на 3D-принтере — проверить сборку до запуска в серию
