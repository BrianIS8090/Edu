# Наружное и архитектурное освещение

Наружное освещение — это одновременно инженерная задача (безопасность, нормы, климат) и дизайнерская (образ здания, атмосфера, акценты). Этот урок покрывает нормы, приёмы, оборудование и типичные ошибки при проектировании наружного и архитектурного освещения.

---

## 1. Классификация наружного освещения

### По назначению

| Тип | Задача | Примеры |
|---|---|---|
| Функциональное | Безопасность, навигация | Улицы, дороги, парковки, дворы |
| Архитектурное | Эстетика, образ здания | Фасады, колонны, карнизы |
| Ландшафтное | Оформление территории | Деревья, дорожки, водоёмы |
| Акцентное | Привлечение внимания | Входные группы, витрины, вывески |
| Праздничное | Временное декоративное | Гирлянды, проекции |

## 2. Нормативные требования

### Улицы и дороги (СП 52.13330)

| Категория | Средняя яркость, кд/м² | Средняя освещённость, лк |
|---|---|---|
| А1 (магистральные) | 1,6 | 20 |
| А2 | 1,2 | 15 |
| Б1 (районного значения) | 1,0 | 15 |
| Б2 | 0,8 | 10 |
| В1 (местные) | 0,4 | 6 |
| В2 | 0,2 | 4 |

### Пешеходные зоны

| Зона | Средняя освещённость, лк |
|---|---|
| Площади, главные пешеходные улицы | 10–20 |
| Второстепенные пешеходные дорожки | 4–6 |
| Дворовые территории | 2–4 |
| Парковки | 10–20 |
| Входы в здания | 20–50 |

### Архитектурное освещение фасадов (СП 52.13330, р. 7)

| Категория здания | Средняя яркость фасада, кд/м² |
|---|---|
| Общегородские доминанты | 10–30 |
| Здания на магистралях | 5–10 |
| Здания на второстепенных улицах | 3–5 |

## 3. Приёмы архитектурного освещения

### Заливающий свет (Flood Lighting)

Освещение фасада целиком прожекторами с земли или с опор.

**Применение:** Памятники архитектуры, крупные общественные здания.

| Параметр | Рекомендация |
|---|---|
| Расстояние от здания | 0,5–1,5 × высота освещаемой части |
| Угол наклона | 30–60° от горизонтали |
| КСС прожектора | Узкая (10–30°) для высоких зданий, широкая (40–80°) для низких |
| Мощность | Зависит от яркости фасада и отражающей способности |

### Локальная подсветка (Accent Lighting)

Выделение отдельных элементов: карнизы, колонны, пилястры, арки.

**Приёмы:**
- **Подсветка снизу (uplighting)** — драматический эффект, тени вверх
- **Подсветка сверху (downlighting)** — естественный эффект, тени вниз
- **Боковой свет (sidelighting)** — подчёркивает рельеф, текстуру

### Контурное освещение (Contour Lighting)

Линейные светильники или LED-ленты по контуру здания.

**Применение:** Современная архитектура, торговые центры, мосты.

| Технология | Особенности |
|---|---|
| LED-лента в профиле | Непрерывная световая линия, IP67+ |
| Линейные LED-светильники | Более мощный и равномерный свет |
| RGB / RGBW | Динамическое изменение цвета |
| Pixel LED | Индивидуальное управление каждым пикселем (медиафасады) |

### Световой карниз / козырёк (Cove Lighting)

Скрытый источник света освещает поверхность отражённым светом.

**Применение:** Подсветка карнизов, навесов, ниш.

### Встроенное в грунт (In-ground)

Светильники вмонтированы в покрытие или грунт, светят вверх.

| Параметр | Требование |
|---|---|
| IP | IP67–IP68 |
| IK | IK10 (если есть пешеходная нагрузка) |
| Несущая способность | До 5000 кг для проезда транспорта |
| Дренаж | Обязательно предусмотреть отвод воды |

## 4. Ландшафтное освещение

### Подсветка деревьев

| Приём | Описание | Светильник |
|---|---|---|
| Uplight (снизу) | Светильник у основания ствола | Грунтовый, узкий луч |
| Downlight (сверху) | Светильник на дереве или рядом | На кронштейне, широкий луч |
| Moonlighting | Мягкий свет через крону | На дереве, направлен вниз |
| Silhouette | Свет за деревом → силуэт | На стене или земле за деревом |

### Подсветка дорожек

- **Столбики (bollards)** — высота 0,5–1,0 м, шаг 3–5 м
- **Встроенные в покрытие** — IP67, IK08+
- **Парковые опоры** — высота 3–4 м, шаг 15–20 м

### Подсветка воды

- **Подводные светильники** — IP68, нержавеющая сталь, низковольтные (12–24 В)
- **Плавающие светильники** — декоративный эффект
- **Подсветка фонтанных струй** — специализированные герметичные светильники

## 5. Выбор оборудования

### Требования к наружным светильникам

| Параметр | Минимум |
|---|---|
| IP | IP65 (под навесом — IP44) |
| IK | IK08 (общедоступные места — IK10) |
| Рабочая температура | –40...+50 °C (для РФ) |
| Материал корпуса | Алюминий (литьё / экструзия) + порошковая окраска |
| Рассеиватель | Закалённое стекло (для грунтовых и антивандальных) |
| Крепёж | Нержавеющая сталь |
| Срок службы | ≥ 50 000 ч (LED) |

### Цветовая температура

| Зона | Рекомендуемая CCT |
|---|---|
| Исторические здания | 2700–3000 К (тёплый) |
| Современная архитектура | 3000–4000 К |
| Дороги и парковки | 4000–5000 К (нейтральный/холодный) |
| Парки и скверы | 2700–3000 К (тёплый, уютный) |
| Промышленные территории | 4000–5000 К |

### Углы раскрытия (КСС)

| Задача | Угол | Обозначение |
|---|---|---|
| Подсветка высоких объектов (башни, колонны) | 6–15° | Очень узкий (VNSP) |
| Акцентная подсветка | 15–30° | Узкий (NSP) |
| Подсветка среднего фасада | 30–50° | Средний (MFL) |
| Заливающий свет | 50–80° | Широкий (WFL) |
| Ландшафтная подсветка | 80–120° | Очень широкий (VWFL) |

## 6. Световое загрязнение

### Проблема

Неправильно спроектированное наружное освещение:
- Засвечивает окна жилых домов
- Создаёт блёсткость для водителей
- Нарушает экосистемы (насекомые, птицы)
- Снижает видимость звёздного неба

### Классификация зон по CIE (E1–E4)

| Зона | Описание | Макс. засветка окон, лк |
|---|---|---|
| E1 | Природные территории | 0 (до 23:00), 0 (после) |
| E2 | Сельская местность | 5 (до 23:00), 1 (после) |
| E3 | Пригород | 10 (до 23:00), 2 (после) |
| E4 | Городской центр | 25 (до 23:00), 5 (после) |

### Способы борьбы со световым загрязнением

1. **Направление света вниз** — не допускать прямого излучения выше горизонта
2. **Козырьки и экраны** — ограничение светового пучка
3. **Тёплая цветовая температура** — ≤ 3000 К (меньше голубого спектра)
4. **Диммирование в ночное время** — снижение яркости после 23:00
5. **Точное распределение света** — использовать оптику под задачу
6. **Full Cut-Off светильники** — нет излучения выше горизонтали

## 7. Электрика для наружного освещения

### Особенности проводки

| Требование | Решение |
|---|---|
| Защита от влаги | Кабель ВВГнг-LS в земле — в ПНД-трубе |
| Глубина прокладки | ≥ 0,7 м (с песчаной подушкой) |
| Защита от замерзания | Без воды внутри светильников (дренаж, герметичность) |
| Грозозащита | УЗИП (устройство защиты от импульсных перенапряжений) |
| Заземление | Контур заземления опор, RE ≤ 10 Ом |
| УЗО | 30 мА для групп наружного освещения |

### Управление наружным освещением

| Способ | Описание |
|---|---|
| Фотореле | Включение по уровню освещённости (сумерки) |
| Астрономический таймер | По времени восхода/заката для данной широты |
| DALI / KNX | Сценарии, диммирование, мониторинг |
| Датчики движения | Для дворовых территорий (экономия) |
| Система управления городским освещением | Централизованное через GSM/LoRaWAN |

## 8. Проектирование — чек-лист

### Фасадное освещение

- [ ] Определена категория здания (яркость фасада по нормам)
- [ ] Выбран приём подсветки (заливающий / локальный / контурный)
- [ ] Учтена отражающая способность фасадного материала
- [ ] Светильники IP65+, рабочая температура –40°C+
- [ ] Предусмотрено электропитание и обслуживание (доступ к светильникам)
- [ ] Проверена засветка соседних зданий и окон
- [ ] Расчёт в DIALux (наружная сцена)

### Территория

- [ ] Освещённость дорожек и парковок по нормам
- [ ] Равномерность ≥ 0,25 (дорожки), ≥ 0,4 (парковки)
- [ ] Столбики / опоры с антивандальной защитой (IK08+)
- [ ] Грунтовые светильники IP67+, с дренажем
- [ ] Кабельные трассы в ПНД-трубах, глубина ≥ 0,7 м
- [ ] УЗИП на вводе щита наружного освещения
- [ ] Управление (фотореле / астротаймер)

## 9. Типичные ошибки

| Ошибка | Последствия | Решение |
|---|---|---|
| Слишком яркий фасад | Засветка окон, жалобы жителей | Проверить нормы CIE по зонам |
| Свет направлен вверх | Световое загрязнение, перерасход энергии | Full Cut-Off, направлять вниз |
| IP54 на улице | Попадание влаги, выход из строя | IP65+ для наружного применения |
| Нет дренажа в грунтовых | Скопление воды, повреждение | Дренажная подушка из гравия |
| Холодный свет (6500 К) у исторических зданий | Неестественный вид, разрушает атмосферу | 2700–3000 К для истории |
| Нет УЗИП | Выход из строя при грозе | УЗИП на каждый щит наружного освещения |
| Недоступность для обслуживания | Замена лампы = автовышка | Продумать доступ на этапе проекта |

## Итоги

- Наружное освещение делится на **функциональное**, **архитектурное** и **ландшафтное**
- Нормы по СП 52.13330 и СП 323.1325800 — обязательны
- Для фасадов: **заливающий**, **локальный** или **контурный** свет
- IP65+, –40 °C, закалённое стекло — минимум для наружных светильников
- **Световое загрязнение** — не забывать про засветку окон и направление света
- **УЗИП**, герметичность, дренаж — ключевые вопросы электрики на улице
