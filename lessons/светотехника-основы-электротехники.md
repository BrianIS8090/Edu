# Основы электротехники для проектировщика

Инженер-проектировщик светильников должен свободно ориентироваться в электрических величинах, схемах и принципах. Этот урок — фундамент, без которого невозможно грамотно проектировать системы освещения.

---

## 1. Электрические величины

### Напряжение (U), вольт [В]

Разность потенциалов между двумя точками цепи. Аналогия — давление воды в трубе.

| Параметр | Значение |
|---|---|
| Бытовая сеть РФ | 230 В (фаза–нейтраль) |
| Линейное напряжение | 400 В (фаза–фаза) |
| Низковольтное освещение | 12 В, 24 В, 48 В DC |
| Безопасное напряжение | ≤ 50 В AC / ≤ 120 В DC |

### Ток (I), ампер [А]

Количество заряда, проходящего через проводник за секунду.

- **Постоянный ток (DC)** — направление не меняется. Используется в LED-модулях, драйверах.
- **Переменный ток (AC)** — направление меняется 50 раз в секунду (50 Гц в РФ). Приходит из розетки.

### Мощность (P), ватт [Вт]

Скорость преобразования электрической энергии.

```
P = U × I
```

Для переменного тока важно различать:

- **Активная мощность (P)** [Вт] — реальная работа (нагрев, свет)
- **Реактивная мощность (Q)** [вар] — энергия, запасаемая в катушках и конденсаторах
- **Полная мощность (S)** [ВА] — S = U × I (то, что «видит» сеть)
- **Коэффициент мощности (PF)** — PF = P / S (для качественных драйверов PF > 0,9)

### Сопротивление (R), ом [Ом]

Противодействие материала протеканию тока.

```
Закон Ома: I = U / R
```

**Практический пример:** LED-лента 24 В потребляет 1 А → сопротивление нагрузки = 24 / 1 = 24 Ом.

## 2. Закон Ома и расчёт цепей

### Закон Ома для участка цепи

```
U = I × R
I = U / R
R = U / I
```

### Последовательное соединение

Элементы включены «цепочкой» — ток один и тот же, напряжения складываются.

```
R_общ = R₁ + R₂ + R₃
U_общ = U₁ + U₂ + U₃
I = const
```

**Где встречается:** LED-модули на токовом драйвере — светодиоды соединены последовательно.

### Параллельное соединение

Элементы включены «лесенкой» — напряжение одно и то же, токи складываются.

```
1/R_общ = 1/R₁ + 1/R₂ + 1/R₃
I_общ = I₁ + I₂ + I₃
U = const
```

**Где встречается:** Светильники в групповой линии — все подключены к одному напряжению 230 В.

## 3. Переменный ток и трёхфазные сети

### Однофазная сеть (230 В)

Стандартное подключение: фаза (L), нейтраль (N), защитное заземление (PE).

- Используется для небольших нагрузок (до ~7 кВт в быту)
- Большинство светильников подключаются к однофазной сети

### Трёхфазная сеть (400 В)

Три фазы (L1, L2, L3), нейтраль (N), заземление (PE).

| Параметр | Значение |
|---|---|
| Фазное напряжение | 230 В (фаза–нейтраль) |
| Линейное напряжение | 400 В (фаза–фаза) |
| Соотношение | U_лин = U_фаз × √3 |

**Зачем проектировщику:** При больших нагрузках (торговые центры, склады, производства) освещение распределяют по трём фазам для равномерной загрузки сети.

### Правило равномерной загрузки фаз

Разница между нагрузками фаз не должна превышать 30%. Пример:

```
Фаза L1: 3,2 кВт (освещение зоны A)
Фаза L2: 2,8 кВт (освещение зоны B)
Фаза L3: 3,0 кВт (освещение зоны C)
→ Перекос: (3,2 - 2,8) / 3,0 = 13% — допустимо
```

## 4. Заземление и защита

### Системы заземления

| Система | Описание | Где применяется |
|---|---|---|
| TN-S | Отдельные N и PE от трансформатора | Новые здания |
| TN-C-S | Совмещённый PEN разделяется на вводе | Реконструкция |
| TN-C | Совмещённый PEN (устаревшая) | Старый жилфонд |
| TT | Местное заземление | Частные дома |

### Классы защиты оборудования

| Класс | Заземление | Изоляция | Примеры |
|---|---|---|---|
| 0 | Нет | Основная | Запрещён в РФ для светильников |
| I | Да (PE) | Основная | Большинство светильников |
| II | Нет | Двойная/усиленная | Светильники со знаком ◻ в ◻ |
| III | Нет | БСНН (≤50 В) | Низковольтные LED-системы |

### Автоматические выключатели

Подбор автомата для группы освещения:

```
I_авт = P_общ / (U × cos φ)
```

**Пример:** 20 светильников × 40 Вт = 800 Вт → I = 800 / (230 × 0,95) ≈ 3,7 А → автомат C6 или C10.

**Важно:** Для LED-драйверов учитывать пусковой ток (inrush current) — он может в 20–60 раз превышать рабочий на доли секунды. Производители драйверов указывают максимальное количество драйверов на один автомат.

## 5. Провода и кабели

### Выбор сечения

Сечение провода зависит от тока нагрузки и длины линии.

| Сечение, мм² | Допустимый ток (медь, в трубе) | Типичное применение |
|---|---|---|
| 1,5 | 15 А | Групповые линии освещения |
| 2,5 | 21 А | Розеточные группы |
| 4,0 | 27 А | Длинные линии освещения |
| 6,0 | 34 А | Магистральные линии |

### Падение напряжения

Для длинных линий освещения критично падение напряжения:

```
ΔU = (2 × L × I × ρ) / S

L — длина линии, м
I — ток, А
ρ — удельное сопротивление меди (0,0175 Ом·мм²/м)
S — сечение провода, мм²
```

**Допустимое падение:** не более 5% от номинального (для 230 В — не более 11,5 В).

**Пример:** Линия 50 м, ток 5 А, сечение 1,5 мм²:
```
ΔU = (2 × 50 × 5 × 0,0175) / 1,5 = 5,8 В (2,5%) — допустимо
```

### Маркировка кабелей для освещения

- **ВВГнг(А)-LS** — стандарт для прокладки в зданиях (негорючий, с низким дымовыделением)
- **NYM** — импортный аналог, допустим для внутренней прокладки
- **ШВВП, ПВС** — только для подключения переносных светильников, не для стационарной проводки

## 6. Практические расчёты

### Расчёт количества светильников на автомат

**Дано:** драйвер Mean Well HLG-150H (150 Вт, PF=0,97, inrush 65 А / 530 мкс)

```
Рабочий ток: 150 / (230 × 0,97) = 0,67 А
Автомат C16: макс. 16 А / 0,67 А = 23 светильника (по рабочему току)
По inrush (данные Mean Well): макс. 12 шт. на C16
→ Принимаем: 12 светильников на автомат
```

### Расчёт сечения кабеля для группы

**Дано:** 10 светильников × 50 Вт, линия 80 м, 230 В

```
P = 10 × 50 = 500 Вт
I = 500 / (230 × 0,95) = 2,3 А → сечение 1,5 мм² (по току — достаточно)
ΔU = (2 × 80 × 2,3 × 0,0175) / 1,5 = 4,3 В (1,9%) — допустимо
→ Принимаем: ВВГнг(А)-LS 3×1,5
```

## Итоги

- **Закон Ома** — основа всех расчётов: U = I × R
- **Мощность** — P = U × I; для AC учитывать cos φ
- **Трёхфазные сети** — равномерно распределять нагрузку по фазам
- **Заземление** — большинство светильников класса I (с PE) или класса II (двойная изоляция)
- **Сечение кабеля** — подбирать по току И по падению напряжения
- **Пусковой ток** — главный фактор при выборе автомата для LED-освещения
