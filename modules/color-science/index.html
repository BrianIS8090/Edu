Что такое CIE 1931 xy?

Диаграмма цветности CIE 1931 — стандартное двумерное представление всех воспринимаемых цветов. Каждая точка (x, y) определяет цветовой тон независимо от яркости. Граница диаграммы — спектральные цвета (монохроматические).

x = X / (X+Y+Z)
y = Y / (X+Y+Z)
z = 1 − x − y
X, Y, Z — трёхстимульные значения CIE
Линия планкова (CCT-линия) — кривая, по которой движется цвет абсолютно чёрного тела при нагреве от ~1000К до >10000К. Белый свет всегда находится вблизи неё.

Зачем нужна u'v' диаграмма?

Пространство CIE xy неравномерно — одинаковые физические различия цветов выглядят разными геометрическими расстояниями. Пространство u'v' (CIE 1976) примерно в 3 раза более равномерно.

u' = 4X / (X + 15Y + 3Z) = 4x / (−2x + 12y + 3)
v' = 9Y / (X + 15Y + 3Z) = 9y / (−2x + 12y + 3)

Δu'v' = √((Δu')² + (Δv')²)
1 SDCM ≈ Δu'v' ≈ 0.001–0.003
SDCM (Standard Deviation of Color Matching) — единица измерения допуска цветности. 1 SDCM — минимально воспринимаемая разница. Промышленный стандарт: 3–5 SDCM для LED.
Изотермы цветовой температуры

На u'v' диаграмме изотермы (линии постоянного ΔT от планковской кривой) перпендикулярны CCT-линии. Это удобно для оценки Duv — отклонения от планковской кривой.

Duv = расстояние от точки до
планковской кривой в u'v'
Duv > 0 → зеленее/выше
Duv < 0 → розовее/ниже

Что такое MacAdam-эллипс?

В 1942 году Дэвид Маккадам установил, что цвета, воспринимаемые как одинаковые, лежат не в точке, а в области — эллипсе вокруг референсного цвета. Размер и форма эллипса меняются по всей диаграмме.

1 SDCM = 1σ эллипса Маккадама
Эллипс 3 SDCM → 3σ → ~99% приборов
в этом допуске неразличимы для глаза

Δu'v'(1 SDCM) ≈ 0.001–0.003
(зависит от CCT и положения)
ANSI C78.377 — стандарт допусков цветности для LED. Определяет квадратные зоны допуска для каждой CCT. Стандарт требует ≤7 SDCM, Premium ≤5, Hi-CRI ≤3.

Ключевые характеристики спектров

LED (белый) создаётся синим чипом (~450нм) + жёлтый люминофор YAG:Ce. Отсюда провал в красной области (~600нм) и пик в синем — «blue spike». Именно синий пик вызывает озабоченность по биодинамике (HEV).

CRI Ra = среднее R1–R8 (основные цвета)
R9 = насыщенный красный (часто низкий у LED)
TM-30: Rf (верность) + Rg (гамут)
CRI LED: типично 80–95+ Ra

Галоген — непрерывный тепловой спектр, идеальный для CRI. НЛВД — линейчатый спектр натрия, низкий CRI (~25). D65 — стандартный «дневной свет», используется как эталон в CRI-расчётах.

Тепловые источники: λ_max = 2898/T мкм·К
T = 2700К → λ_max = 1073 нм (ИК)
Люминофорные: пики при фикс. λ
Газоразрядные: линии спектра атомов

🌈 CIE 1931 и трихроматизм

Глаз содержит три типа колбочек (S, M, L), каждый чувствителен к своей части спектра. Любой цвет описывается тремя числами X, Y, Z.

Y = яркость (luminance)
x = X/(X+Y+Z)
y = Y/(X+Y+Z)
  • —Подкова диаграммы = монохромат. λ = 380–700нм
  • —Прямая линия пурпура — нет монохроматич. аналога
  • —Белая точка E: x=y=1/3 (равная энергия)
  • —D65: x≈0.3127, y≈0.3290

🌡 CCT и планков. кривая

Коррелированная цветовая температура (CCT) — температура абсолютно чёрного тела, чей цвет наиболее близок к цвету источника.

CCT по Маккэми (1992):
CCT = 449n³ + 3525n²
+ 6823.3n + 5520.33
n = (x−0.332)/(y−0.1858)
  • —Duv > 0 → зеленоватый оттенок
  • —Duv < 0 → розоватый/пурпурный
  • —|Duv| < 0.006 — в норме
  • —Типичные LED: ±0.003–0.006

🎯 MacAdam и SDCM

MacAdam (1942) провёл визуальные эксперименты и определил зоны равного восприятия — эллипсы. SDCM нормирует это в статистические отклонения.

SDCM = (Δu'v') / σ(u'v')
σ(u'v') ≈ размер эллипса 1σ
3 SDCM: 99% людей не видят разницы от центра эллипса
  • —1 SDCM ≈ just noticeable difference (JND)
  • —ANSI C78.377: 7-шаговые квадранты
  • —При группировке светильников: ≤3 SDCM
  • —Бин (bin): производитель сортирует LED по SDCM

📊 CRI и TM-30

CRI Ra — индекс цветопередачи. Измеряет точность воспроизведения 8 стандартных цветов относительно референсного источника.

Ra = (R1+R2+...+R8) / 8
R9 — насыщ. красный (важен!)
TM-30: Rf (верность 0–100)
+ Rg (насыщенность 60–140)
  • —Ra ≥ 80: общее освещение
  • —Ra ≥ 90: торговля, медицина, музеи
  • —R9 ≥ 50: тёплые тона передаются верно
  • —Галоген: Ra=100, R9=100 (эталон)

💡 Практика бининга LED

Производители LED сортируют (bins) кристаллы по CCT и цветности. Правильный выбор бина определяет SDCM.

  • —1 bin ≈ ±75К по CCT — ~2–3 SDCM
  • —Смешение бинов в одном объекте → видимая разница
  • —Заказывайте LED одной партии (lot)
  • —При замене: измерьте CCT колориметром
  • —Nichia/Samsung: бины по 50–100К

🔬 Метамеризм

Метамеризм — два предмета выглядят одного цвета при одном источнике и разного при другом. Критически важно для музеев, текстиля, живописи.

Индекс метамеризма MI:
0 — нет метамеризма
>1 — заметный, >2 — сильный
Галоген → LED: MI часто высокий
  • —High-CRI LED минимизируют метамеризм
  • —LED с красным чипом (RGBW) — лучше всего
  • —Для критичных применений: TM-30 Rf > 90
